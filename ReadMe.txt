Игра 21. (Alpha)
Клиен-серверное приложение .Net. 
Для запуска требуется запустить сервер, далее подключаются игроки. Каждые 2 игрока начинается игра и клиентам высылается инструкция.
По правилам игроки ходят по очереди. Имеется подсчет очков. 

Server находится в FinalGame21. 
Реализован файл Unit теста.
